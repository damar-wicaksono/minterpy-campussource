Barycentric Operator¶

class minterpy.schemes.barycentric.operators.BarycentricOperator(transformation: TransformationABC, transformation_data)¶

Bases: minterpy.core.ABC.operator_abstract.OperatorABC

Base class for the barycentric transformation operators.

Specialized versions (dict, factorised, and piecewise) are derived from this base class.
