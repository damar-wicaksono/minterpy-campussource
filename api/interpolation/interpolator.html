Interpolator class¶

class minterpy.interpolation.Interpolator(spatial_dimension: int, poly_degree: int, lp_degree: int)¶

Bases: object

The construction class for interpolation.

Data type which contains all relevant parts for interpolation and caches them.

Attributes
spatial_dimensiondimension of the domain space.
poly_degreedegree of the interpolation polynomials
lp_degreedegree of the \(l_p\) norm used to determine the poly_degree.
multi_indexlexicographically complete multi index set build from (spatial_dimension, poly_degree, lp_degree).
gridGrid instance build from multi_index.
__call__(fct: Callable) → Optional[minterpy.polynomials.newton_polynomial.NewtonPolynomial]¶

Interpolate a given function.

Builds a NewtonPolynomial which interpolates the given fct, where the precomuted setting of the current instance is used.

Parameters

fct (Callable) – Function to be interpolated. Needs to be (numpy) universal function which shall be interpolated. If arr is an np.ndarray with shape arr.shape == (N,spatial_dimension), the signature needs to be fct(arr) -> res, where res is an np.ndarray with shape (N,).

Returns

Interpolation polynomial in Newton form, which interpolates the function fct, where the used divided difference scheme is build from self.multi_index and self.grid.

Return type

NewtonPolynomial

Raises

InterpolationError – Raised if anything goes wrong with the interpolation.
